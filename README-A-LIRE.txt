COMMENT MODIFIER LES PRODUITS

Votre mère doit modifier uniquement le fichier : produits.json

Pour changer un produit, elle modifie simplement :
- nom
- prix
- matiere
- description
- image
- stock

Exemple :
"prix": "14,90 €"

Pour changer la photo :
1. Ajouter la photo dans le dossier images
2. Mettre son nom dans produits.json
Exemple :
"image": "mon-bijou.jpg"

IMPORTANT :
- Ne pas supprimer les guillemets
- Ne pas oublier les virgules entre les produits
- Les photos doivent être dans le dossier images
